The API Showcase is still a work in progress, thus comments and documentation may be lacking.

This web application is a collection of different API functionalities including:

- Generating a variety of graph types similar to what is available in MyKeynote
- A sample NOC type dashboard
- Custom daily reports for TxP, ApP, MWP (7 day and last 4 weeks)
- Slot Alarm configuration

The application is built using PHP 5, Zend framework and Twitter Bootstrap.
