The API Showcase is still a work in progress, thus comments and documentation may be lacking.

This web application is a collection of different API fucntionality including:
-Generating a variety of graph types similar to what is available in MyKeynote
-A sample NOC dashboard
-Custom reports for TxP, ApP, MWP, and Alarm configurations

The application is built using PHP and the ZEND framework, as well as Twitter Bootstrap.
